# Codex Archive

This folder stores long-horizon doctrine artifacts that are worth preserving
without treating them as active runtime truth.

Each codex should be archived in two forms:

- raw source artifact
- normalized metadata record

Registry:

- `CODEX_INDEX.json`

Rules:

- codexes may inform future direction
- codexes do not override `_docs/ARCHITECTURE.md`
- codexes do not imply active implementation
- codexes should state what they are not allowed to claim today
